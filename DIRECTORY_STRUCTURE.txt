================================================================================
                        KK_Lab 工程目录结构说明
================================================================================

KK_Lab 是一个基于 PySide6 (Qt) 的 BES 芯片功耗测试与调试桌面工具，通过
VISA / 串口 / USB-I2C 协议控制多种实验室仪器（电源分析仪、示波器、温箱等），
实现自动化数据采集、PMU 测试、Charger 测试、功耗测试、自定义测试序列和
波形展示等功能。

================================================================================
  分层与规则速查
================================================================================

分层调用关系（铁律）：

    main.py → ui/ ←→ core/ → instruments/ → lib/
                   ↑
             log_config / debug_config / chips

- ui/          禁止阻塞 IO，必须通过 core/ 或 QThread 调用仪器
- instruments/ 禁止依赖 PySide6 Qt Widget / UI 模块
- 日志统一使用 log_config.get_logger(__name__)，禁止 print()
- 仪器统一通过 instruments/factory.py 创建，禁止直接 new 具体仪器
- 新仪器必须同时提供 Mock 实现（instruments/mock/mock_instruments.py）
- 资源路径统一通过 ui/resource_path.py::get_resource_base() 获取，
  兼容源码运行与 PyInstaller 冻结环境
