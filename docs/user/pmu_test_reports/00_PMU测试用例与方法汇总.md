## 3. 通用测试方法（按测试项归类）

### 3.1 上电 / 开关机 / 复位类
- **关机漏电**：扣除分压电阻电流后计净漏电（例：1811 在 4V 对应约 2uA）。
- **开机波形 / 开机 Vbat 瞬态电流**：示波器抓 VBAT、各路输出上电时序与瞬态峰值电流（1811 仿真+实测对比，峰值约 60mA）。
- **RST 电流及波形**：记录 RST 高电平持续时间、低电平恢复时间（1307PH 高 9.911ms / 恢复 14.345ms；2720 RST 生效后 8.86ms 复位）。
- **关机波形**：分 `soft poweroff` 与 `快速关机 + soft poweroff`；快速关机显著降低掉电时间（1307PH 7s → 8ms）。
- **UVLO / 开关机电压**：VBAT 突变（如 4V → 0.5~2V）观察是否正常关机；关机电压可调，开机电压不可调（2720）。
- **PIN 脚耐压**：PWRON/RST/VBAT/VCHG-R/CHG_IN 各自耐压（1811 多为 5V，CHG_IN 16V）。
- **PATTERN 复位**：VCHG_R 输入特定 pattern，确认三路电复位（08/08H 不支持）。
- **Latch Up**：VBAT/VSYS/AC_IN 快速上电 20~50 次，确认无闩锁。
- **开关机撞沿**：AC_ON 给方波同时反复写 soft_power_off，验证开/关机鲁棒性（已知关机异常依赖 WDT 重启）。

### 3.2 电源质量类（DCDC / LDO）
- **带载能力 & 纹波**：normal / sleep(dsleep) 分别测；区分 PWM / BURST / ULP 模式；SIMO 因输出切换纹波偏大（1307PH BUCK_VCORE/VHPPA）。
- **PSRR / 纹波**：LDO、DCDC 分别测电源抑制比与输出纹波。
- **线性调整率 & 负载瞬态响应**：负载 1~100mA 或 1~200mA 在 10/100/1000Hz 切换；输入 3.2~4.2V 多频率瞬态。
- **DCDC 效率**：调试初期用 LDO→DCDC 配置初测，芯片可休眠后用休眠 bin、BUCK 切 normal mode 复测；被测路 1~200mA ramp，另一路带 5mA。
- **输出电压挡位扫描**：遍历各 LDO/DCDC 全挡位寄存器，记录输出电压范围、步进、线性度。
- **掉坑 / 过冲**：电压 ramp 切换（ramp_en/ramp_step）观察过冲（稳定复现）与掉坑（概率复现）。
- **BG 切换压差**：normal/sleep 大小 BG 切换导致同 Vbit 输出不一致，需调低 lp_bias。

### 3.3 时钟 / ADC / 接口类
- **GPADC**：精度（输入 0 时为 offset）、稳定性（timer=2000 counts，2000~10000 次，误差 ≤ ±10mV 为 pass）、高低温一致性；需关注 FT 校准与 burst mode。
- **LPO**：默认频率、频率范围、JITTER、100 次软件频偏、高低温、休眠唤醒频偏。
- **外部 32K**：Capbit 对应频率、静态功耗、高低温、PMU 重载对频率影响。
- **单线通讯**：测不同波特率（921600/1152000/2000000）在 3.3V/1.8V 的通信稳定性（高波特率需加强外部上拉）。
- **EFUSE 读写**：按 WRITE_BITS / READ 脚本验证读写（1811 报告附完整脚本）。

### 3.4 GPIO / 其它
- **GPIO 开机脉冲**：开机时电压 < 50mV 且无毛刺判 pass。
- **GPIO 驱动能力 / 阻抗**：记录输出阻抗、输入阻抗、上下拉。
- **SPK 短路保护**：ocp_sel，限流持续约 1ms；注意最低档误触风险。
- **VMIC 底噪**：与坏片/其它芯片横向比较是否满足使用标准。
- **Load Switch 导通阻抗**：逐路（VUSB/VMEM/VEXT2/VCORE_L1/VANA_IN）测导通阻抗。
