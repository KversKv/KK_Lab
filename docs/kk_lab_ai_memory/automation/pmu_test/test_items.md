## T-20260626-150101 - 开机波形与开机 Vbat 瞬态电流

- 页面：automation/pmu_test
- 目标：验证上电时序正确、各路输出按序建立、开机峰值电流在预期内
- 前置条件：
  - VBAT 接电池/电源，示波器接 VBAT 与各路输出
- 参数：
  - vbat: 典型 3.2~4.2V
- 步骤：
  1. 上电触发开机
  2. 抓 VBAT 与各路输出上电时序
  3. 记录开机瞬态峰值电流
- 期望结果：
  - 时序正确，峰值电流在设计预期内（如 1811 约 60mA）
- 数据记录：
  - 开机波形、峰值电流值
- 适用范围：
  - 所有 PMU 芯片

## T-20260626-150102 - 带载能力与纹波（normal / sleep）

- 页面：automation/pmu_test
- 目标：验证各路 DCDC/LDO 最大带载与纹波满足规格
- 前置条件：
  - 选定工作模式（PWM/BURST/ULP），配置对应 bin
- 参数：
  - load: 1~200mA（DCDC）/ 1~150mA（LDO）
  - mode: normal / dsleep
- 步骤：
  1. 逐路加载到额定最大电流
  2. 记录纹波（含/去过冲）与跌落
- 期望结果：
  - 达额定带载不掉压、纹波满足规格
- 数据记录：
  - 各路最大带载、纹波 mVpp、Drop
- 适用范围：
  - 所有 PMU；SIMO 注意切换过冲（见 lessons L-20260626-150201）
- 关联项：
  - lessons: L-20260626-150201, L-20260626-150203

## T-20260626-150103 - DCDC 效率

- 页面：automation/pmu_test
- 目标：测各路 BUCK 在负载范围内的转换效率
- 前置条件：
  - 调试初期用 LDO→DCDC 配置初测；可休眠后用休眠 bin、BUCK 切 normal mode 复测
- 参数：
  - load: 1~200mA ramp（被测路），另一路带 5mA
- 步骤：
  1. 被测 BUCK 输出加 1~200mA ramp 负载
  2. 记录输入/输出功率算效率
- 期望结果：
  - 效率曲线符合 datasheet
- 数据记录：
  - 负载-效率表
- 适用范围：
  - 所有含 DCDC 的 PMU
- 关联项：
  - 页面 pmu_dcdc_efficiency

## T-20260626-150104 - 各 LDO/DCDC 输出电压所有挡位扫描

- 期望结果：
  - 范围/步进/线性度满足规格
- 数据记录：
  - 挡位→电压→步进→线性度表
- 适用范围：
  - 所有 PMU
- 关联项：
  - 页面 pmu_output_voltage

## T-20260626-150105 - 负载快速变化瞬态响应

- 页面：automation/pmu_test
- 目标：验证负载阶跃下输出过冲/恢复
- 前置条件：
  - 电子负载支持 10/100/1000Hz 切换
- 参数：
  - load: DCDC 1~200mA / LDO 1~100mA
  - freq: 10 / 100 / 1000 Hz
- 步骤：
  1. 负载在两点间按各频率切换
  2. 抓输出瞬态
- 期望结果：
  - 过冲/下冲与恢复时间满足规格
- 数据记录：
  - 各频率瞬态波形
- 适用范围：
  - 所有 PMU

## T-20260626-150106 - GPADC 精度/稳定性/高低温

- 页面：automation/pmu_test
- 目标：验证 GPADC 各通道精度、稳定性与温度一致性
- 前置条件：
  - FT 校准完成；burst mode 配置；timer=2000 counts
- 参数：
  - vref: 1.5V
  - repeat: 2000~10000 次
- 步骤：
  1. 各通道输入已知电压
  2. 多次读数统计误差
  3. 高低温复测
- 期望结果：
  - 误差 ≤ ±10mV 为 pass
- 数据记录：
  - 各通道误差、稳定性曲线
- 适用范围：
  - 所有 PMU
- 关联项：
  - 页面 pmu_gpadc

## T-20260626-150108 - 时钟相关（LPO / 外部 32K）

- 页面：automation/pmu_test
- 目标：验证 LPO 频率范围/JITTER/频偏与外部 32K Capbit/重载影响
- 前置条件：
  - 频率计/示波器
- 参数：
  - capbit: 默认 9'b1_0010_0000（对应约 32.7646kHz）
- 步骤：
  1. 测 LPO 默认频率/范围/JITTER/100 次频偏/高低温
  2. 测 32K Capbit、静态功耗、重载对频率影响
- 期望结果：
  - 频率范围与频偏满足规格
- 数据记录：
  - LPO 频率范围、JITTER、32K 频率
- 适用范围：
  - 含 LPO/32K 的 PMU
- 关联项：
  - 页面 pmu_clk

## T-20260626-150109 - 可靠性（Latch Up / LX 负压 / 撞沿）

- 页面：automation/pmu_test
- 目标：验证上电闩锁、LX 负压、开关机撞沿鲁棒性
- 前置条件：
  - 可快速上电与注入负压、方波
- 参数：
  - latchup_count: 20~50 次
- 步骤：
  1. VBAT/VSYS/AC_IN 快速上电多次查 Latch Up
  2. LX 断开电感注入负压查保持电流
  3. AC_ON 方波 + 反复 soft_power_off
- 期望结果：
  - 无 Latch Up；撞沿不死机（异常依赖 WDT）
- 数据记录：
  - 测试次数与现象
- 适用范围：
  - 建议所有 PMU 必测
- 关联项：
  - lessons: L-20260626-150206
