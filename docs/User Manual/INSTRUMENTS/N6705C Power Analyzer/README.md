# N6705C Power Analyzer

本导航项是 **Keysight N6705C 直流电源分析仪**的总入口，包含两个子页面（通过悬停按钮右侧箭头展开子菜单切换）：

| 子页面 | 用途 |
|---|---|
| [N6705C Analyser](./N6705C%20Analyser.md) | 实时控制台：电压/电流设置、通道开关、Auto Set、批量设置、功耗读取 |
| [N6705C Datalog](./N6705C%20Datalog.md) | 长时间数据采集与波形分析（含导入导出、Marker、统计） |

## 双插槽（A / B）模型

N6705C Analyser 支持同时连接两台 N6705C（插槽 A 与 B），实现双仪器对比测量或四通道以上同步采集；Datalog 同样支持双仪器叠加显示并支持时间偏移对齐。

## 切换方式

1. 点击导航栏 `N6705C` 按钮右侧的 ▶ 箭头展开子菜单。
2. 选择 `N6705C Analyser` 或 `N6705C Datalog`。
3. 子菜单选中状态会保留，下次进入直接回到上次子页。

## 共用仪器连接

两个子页面共用 `n6705c_top`（N6705CTop）维护的连接会话：在 Analyser 中连接的设备会自动同步到 Datalog，反之亦然，无需重复搜索/连接。
