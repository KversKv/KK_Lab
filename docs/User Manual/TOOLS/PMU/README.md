# PMU Tools

PMU（Tools 分组）是**离线 PMU 配置工具**集合，针对具体芯片型号提供图形化配置界面，用于直接读写 PMU 内部寄存器、查看模块拓扑、调试 LDO/BUCK 配置。

> 与 AUTOMATION 下的 `PMU Test` 不同：PMU Test 是自动化测试套件（测指标），PMU Tools 是手动配置工具（调寄存器）。

## 子页面索引

| 子页面 | 芯片型号 | 当前状态 |
|---|---|---|
| [1811](./1811.md) | BES1811 | 已实现（完整拓扑 + I2C 控制） |
| [1860](./1860.md) | BES1860 | 占位（待实现） |

## 页面入口

- 导航栏 → `TOOLS` → `PMU` → 悬停展开子菜单选择芯片型号。
- 对应源码：`ui/pages/pmu/`，子文件夹 `pmu_1811/` 与文件 `pmu_1860_ui.py`。

## 共用约定

- **驱动方式**：通过 USB-I2C（设备地址 0x17，10 位寄存器地址，16 位数据）直接读写 PMU 寄存器。
- **I2C 即用即销**：每次操作自建/销毁 I2C 接口，避免跨线程共享问题。
- **QThread Worker**：所有 I2C 操作通过 `workers.py` 的 Worker 异步执行，UI 不阻塞。
- **芯片寄存器表**：从 `chips/bes1811_pmu.py` 等芯片配置文件读取，不在代码中硬编码。
