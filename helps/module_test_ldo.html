Module Test — LDO（线性稳压器模块测试）

页面介绍

LDO 子页面用于对线性稳压器（LDO）执行完整可测测试项，覆盖输出电压、调整率、静态电流与交流特性，一次执行产出合并报告。

测试项

  • 输出电压扫描 — 遍历 DAC 挡位，记录各挡输出电压（N6705C）
  • 负载调整率 — 1~200mA 扫描，计算 ΔVout/ΔIload（N6705C）
  • 线性调整率 — Vin 3.2~4.2V 扫描，计算 ΔVout/ΔVin（N6705C）
  • 静态电流 — 各模式 Iq 测量（N6705C）
  • 输出纹波 — Vpp / RMS（示波器，未接示波器自动跳过）
  • 电源抑制比 PSRR — dB @ 频点（示波器）
  • 负载瞬态响应 — 过冲/下冲/恢复时间（示波器）

使用方法

  1. 在仪器连接区连接 N6705C（必接）；如需测纹波/PSRR/瞬态项，再连接示波器。
  2. 在被测配置区填写芯片名称、操作员、Vin/Vout/Iload 通道映射、Vout 标称值与 DAC 范围。
  3. 在测试项清单勾选要执行的项（未接示波器时示波器项自动灰化并提示）。
  4. 在统一参数区设置负载范围、Vin 扫描范围、PSRR 频点、瞬态频率。
  5. 点击「开始测试」，执行日志区实时刷新进度与逐项结果；可随时「停止」。
  6. 测试完成后点击「打开报告」查看 HTML 汇总报告（含逐项 PASS/FAIL 表与测量数据）。

结果与报告

结果落盘于 Results/module_test/ldo/<芯片>_<时间戳>/，每项一份 CSV，汇总为 report.html。
