Custom Test - 自定义测试序列编辑器

概述

Custom Test 是一个可视化自定义测试序列编辑器，类似 Keysight BenchVue 的 Bench 功能。 您可以通过拖拽方式自由组合仪器控制和逻辑节点，构建完整的自动化测试流程。

界面布局

  • 左栏：仪器连接区 + 节点面板（双击或拖拽添加节点）
  • 中栏：序列画布（树形结构展示测试流程）
  • 右栏：属性面板（编辑选中节点的参数）
  • 底部：执行日志 / 实时数据表 / 图表

节点类型

仪器节点

  • Chamber Set Temp：设置温箱目标温度，可选等待稳定
  • N6705C Set Voltage：设置 N6705C 通道电压和限流
  • N6705C Measure：测量电压/电流/功率，结果自动写入变量池
  • Scope Measure Freq：示波器频率测量
  • Scope Measure：示波器通用测量（Vpp/Vrms/Mean 等）
  • RF Analyzer Measure：综测仪测量（预留接口）

逻辑节点

  • Loop (Range)：范围循环（起始/终止/步进），子节点在循环体内执行
  • Loop (List)：列表循环（离散值），支持温度点列表等场景
  • If / Else：条件分支，支持表达式判断
  • Set Variable：变量赋值
  • Delay：延时等待
  • Math Expression：数学运算

数据节点

  • Record Data：记录一行数据到结果集
  • Export Result：导出结果为 CSV 或 Excel

变量系统

所有参数支持 ${变量名} 占位符语法。循环节点会自动注入迭代变量，测量节点会自动写入结果变量。

例如：${temp}、${N6705C_CH1_power}、${scope_CH1_freq}

使用步骤

  1. 在左栏连接所需仪器
  2. 从节点面板双击或拖拽节点到序列画布
  3. 在右栏属性面板中配置参数
  4. 点击 ▶ Run 执行序列
  5. 在底部查看日志、实时数据和图表
  6. 执行完成后自动导出 CSV 到 Results/custom_test/ 目录

模板

可通过 Load 按钮加载内置模板：

  • CASE1: 功耗 vs 温度曲线
  • CASE2: GPADC vs 温度+电压多维测试
  • CASE3: 时钟频率 vs 温度
  • CASE5: 时钟频率 vs 电压

序列保存/加载

使用工具栏的 💾 Save / 📂 Load 按钮保存和加载 JSON 格式的序列文件。
