Charger — Status Register Test（状态寄存器测试）

页面介绍

Status Register Test 页面用于测试 Charger 模块的状态寄存器，验证在不同充电状态下状态寄存器的标志位是否正确反映当前状态。

测试项

  • 状态位验证 — 读取并验证各状态寄存器位的正确性
  • 状态转换测试 — 触发不同充电状态，验证状态位的切换
  • 中断标志测试 — 验证中断标志位的置位和清除机制

测试方法

  1. 连接 N6705C 电源和 I2C 接口
  2. 配置充电器初始工作状态
  3. 设置需要测试的状态寄存器
  4. 点击开始测试，系统自动执行状态转换并读取寄存器
  5. 查看各状态位的测试结果，确认与预期一致
