Charger — Config Traverse Test（配置遍历测试）

页面介绍

Config Traverse Test 页面用于自动遍历 Charger 模块的所有寄存器配置组合，验证各配置下充电器的工作状态和参数是否符合预期。

测试项

  • 寄存器遍历 — 自动遍历所有可配置的寄存器值
  • 配置验证 — 写入寄存器后回读验证，确保配置生效
  • 功能验证 — 在不同配置下测量实际充电参数（电压、电流）

测试方法

  1. 连接 N6705C 电源为 DUT 供电
  2. 通过 I2C 连接到 DUT 的 Charger 模块
  3. 设置遍历范围和步进参数
  4. 点击开始测试，系统自动遍历配置并记录结果
  5. 测试完成后查看结果，确认所有配置项均通过验证
